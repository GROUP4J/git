## git命令
### 查看分支结构
    git log --graph --decorate --oneline --[simplify-merges|full-history|simplify-by-decoration] --all

### 创建新分支
	git branch [branch_name]

### 切换分支
	git checkout [branch_name]

### 显示分支
	git branch [-a]

### 查看远程分支
	git branch -r

### 删除本地分支
	git branch -d [branch_name]

### 删除远程分支
	git push origin:[branch_name]
